<h1>Brazilian E-Commerce SQL Analysis using DuckDB</h1>
<hr>
This project explores 100K rows of e-commerce transactions in Brazil using DuckDB.

<h3>Dataset</h3>
<hr>
Download from Kaggle: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/data

<h3>Files</h3>
<hr>
brazil_ecom.db — the DuckDB file you can open with duckdb brazil_ecom.db --ui <br>
data/ — contains the raw CSV files from Kaggle
